## Executive Summary

Daedalus is a Python application named `rsps-crewai-team` that combines a CrewAI planning workflow, an OpenClaw-backed coding worker, cron automation, Git/GitHub helpers, and a static HTML/CSS/JavaScript dashboard for supervising work orders. The package requires Python `>=3.10,<3.14`, depends on `crewai[litellm,tools]` and `python-dotenv`, and is built with Hatchling.
